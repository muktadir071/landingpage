# Leaf & Legacy - Tea Brand Login Page

A beautifully designed, interactive login and signup landing page for **Leaf & Legacy**, a premium tea brand from the highlands of Panchagarh, Bangladesh.

## 🌿 Features

- **Interactive Design**: Pull the cord on the lamp to illuminate the page and activate animations
- **Animated Leaves**: Falling leaf particles create an immersive garden atmosphere
- **Login & Signup**: Dual-panel forms with smooth transitions
- **Responsive Layout**: Adapts seamlessly to all screen sizes
- **Elegant Aesthetics**: Dark green color scheme with golden accents and serif typography
- **Form Interactions**: Real-time personalized welcome messages

## 🎨 Design Highlights

- Custom CSS animations and transitions
- Interactive lamp with physics-based cord mechanics
- Ambient glowing effects and noise textures
- Graceful form state management
- Mobile-friendly touch interactions

## 🚀 How to Use

1. Open `index.html` in your browser
2. Pull the cord on the lamp to light it up and reveal the login card
3. Choose to log in or sign up
4. Fill in the form fields
5. Click submit to see a personalized welcome message

## 📝 Form Fields

**Login:**
- Email
- Password

**Signup:**
- Full Name
- Email
- Password

## 🌱 About Leaf & Legacy

Leaf & Legacy specializes in pure, premium tea sourced directly from the tea gardens of Panchagarh, Bangladesh. Our landing page combines elegant design with interactive storytelling to create a memorable user experience.
